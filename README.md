# CourseWork1_WenHsi
## Frontend 
1. Get into "frontend" directory. <br>
```cd foodhealthtrend/frontend```
2. Install all node_modules. <br>
```npm install```
3. Activate frontend. <br>
```npm start```
## Backend
1. Get into "server" directory. <br>
``` cd foodhealthtrend/server ```
2. Run sqlite.py to load dataset into database. <br>
```python sqlite.py```
3. Activate server. <br>
```python app.py```
## API Documentation
- API Documentation is in the form of Postman.json
- You can import in Postman app.
