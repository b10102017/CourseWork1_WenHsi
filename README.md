# CourseWork1_WenHsi
## Frontend 
1. Get into "frontend" directory. <br>
```cd foodhealthtrend/frontend```
2. Install all node_modules. <br>
```npm install```
3. Activate frontend. <br>
```npm start```
## Backend
1. Run sqlite.py to load dataset into database. <br>
```python sqlite.py```
2. Activate server. <br>
```python app.py```
